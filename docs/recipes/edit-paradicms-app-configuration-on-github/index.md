# Recipe: edit the Paradicms app configuration on GitHub

### Navigate to the *Code* tab of your GitHub repository

![Screenshot of GitHub code tab](../github-code-tab.png)

### Click on `app-configuration.ttl`

![Screenshot of viewing the app configuration in GitHub](view.png)

### Click the edit (pencil) button

![Screenshot of the edit button in GitHub](../github-edit-file-button.png)

### Edit the app configuration as needed

![Screenshot of editing the app configuration in GitHub](edit.png)

### Commit changes

When you're done editing, scroll to the bottom of the screen and click the *Commit changes* button.

![Screenshot of commit changes to the app configuration in GitHub](commit.png)
