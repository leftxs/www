# Recipe: edit the configuration of a Paradicms GitHub Action static site generation workflow

### Navigate to the *Code* tab on your GitHub repository

![Screenshot of GitHub code tab](../github-code-tab.png)

### Select the `.github/workflows` folder

![Screenshot of the .github/workflows folder](github-workflows-folder.png)

### Select the `build.yml` file

![Screenshot of the build.yml file](view.png)

### Edit the file by clicking the pencil button.

![Screenshot of the edit button](../github-edit-file-button.png)

### Edit the YAML as needed

![Screenshot of editing the GitHub Actions workflow in GitHub](edit.png)

### Commit changes

When you are done editing, click the *Start commit* button on the right, then the *Commit changes* button.

![Screenshot of commit changes to the GitHub Actions workflow](commit.png)
